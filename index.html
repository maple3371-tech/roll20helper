포스타입 백업 (Word 저장)
포스타입 게시글을 이미지까지 포함해 Word(.doc) 또는 HTML 파일로 저장하는 크롬 확장 프로그램입니다.
설치 방법 보기
  1. 아래 버튼으로 zip 파일을 내려받고 압축을 풉니다.
  2. Chrome 주소창에 chrome://extensions 를 입력합니다.
  3. 우측 상단의 개발자 모드를 켭니다.
  4. 압축해제된 확장 프로그램을 로드를 눌러 압축을 푼 폴더를 선택합니다.
  5. 포스타입 게시글에서 확장 아이콘을 누르면 저장됩니다.
